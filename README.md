Node version requirement 
```
version >= 16.xx.x
```

1. Install packages
```
yarn
```

2. Environmental variables
- create a '.env' file in the source directory
```env
# located in ./.env
# env format

token = "Discord bot token"
GUILD_ID = "Your guild ID here"
```

3. Start Application
- To start as developer
(Uses ts-node-dev)
```
yarn run start:dev
```
- Start locally
```
yarn start
```
